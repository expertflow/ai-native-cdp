## Google Chat Space — AI-Native CDP Project

**Space URL:** https://chat.google.com/room/AAAAyffa-Sk

This is the primary communication channel for the AI-Native CDP project. The team discusses and reviews progress, shares updates, and coordinates here.

**How to apply:** Whenever a PR is submitted, a significant change is made, or a noteworthy update is ready, send a message to this Google Chat space using the `gws-chat-send` skill. Keep messages concise: what changed, why, and a link if applicable (e.g. PR URL, file, commit).
